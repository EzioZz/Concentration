

练习了MVC

Model, View, Controler

- Model: 一般用于封装数据和操纵数据的方法。model中数据的变化一般通过刷新的机制，在view上显示，为了能显示对应的数据，view要先去model中注册。
- View: 可以理解为UI界面
- Controler: 起到不同层面的组织作用，用于控制应用程序的流程。用于处理事件并作出改变。
